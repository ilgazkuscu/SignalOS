# Operator Runbook

1. Load structured evidence with factuality labels.
2. Run the scoring pipeline.
3. Inspect contradiction penalty and hypothesis confidence before acting.
4. Read the narrative output and verify that speculation remains explicitly labeled.
5. Compare thesis probability with market price before using the trade decision layer.

Trip delay is reported; cancellation is NOT confirmed.

The trade decision layer is a disciplined execution aid. It does not replace expert judgment.
